# MDVRP Heuristic Optimization

## Description
This project implements solutions for the Multi-Depot Vehicle Routing Problem (MDVRP) using:

- Greedy Nearest Neighbor (Baseline)
- A* Search with MST Heuristic (Enhancement)

## Setup

```bash
pip install -r requirements.txt

Run Tests
python tests/test_algorithms.py

Features
Modular implementation
Distance matrix optimization
MST-based admissible heuristic
Performance comparison framework
Results

A* improves route quality but increases computation time.
